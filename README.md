## Setup

This project is built using Vite + TypeScript + pnpm + P5.js + Tone.js.

- Install `pnpm` (https://pnpm.io/installation)
- Run `pnpm install` in the root project folder
- Run `pnpm dev`
- Local server should start at `http://localhost:5173`
